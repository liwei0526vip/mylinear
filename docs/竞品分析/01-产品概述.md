# 产品概述

**Linear** 是一款面向现代软件研发团队的项目管理工具，核心理念是**速度、简洁、克制**。覆盖产品开发全生命周期：从路线图到发布（Intake → Plan → Build → Reviews → Monitor）。

---

## 核心定位与差异化壁垒

| 竞品维度 | Linear 的选择 | 解决的问题 |
|---------|-------------|-----------|
| 性能体验 | Local-First 架构（IndexedDB + 增量同步） | 消除网络延迟，离线可用，UI 零延迟 |
| 信息密度 | 三栏布局 + Peek 预览 + 快捷键驱动 | 减少页面跳转，提升信息获取效率 |
| 工作流 | 自定义状态机 + Git 自动流转 | 打通代码提交与任务管理的断裂 |
| AI 生态 | Agent 平台 + MCP 协议 + 6+ AI Agent | 让 AI 成为团队的"虚拟成员" |
| 定价策略 | Free 无限成员（250 活跃 Issue 限制） | 降低入门门槛，通过功能限制推动升级 |

## Linear 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | React + Next.js + TypeScript |
| 状态管理 | MobX（本地优先架构，内存对象图） |
| 本地存储 | IndexedDB（客户端持久化） |
| 后端 | Node.js + TypeScript |
| API | GraphQL + WebSocket |
| 数据库 | PostgreSQL + Redis |
| 基础设施 | Google Cloud + Kubernetes |

### 核心架构特色：本地优先（Local-First）

Linear 的极致性能来自「本地优先架构」，而非前端框架选择：

- **零延迟**：所有操作先写入本地 → UI 即时反馈 → 后台异步同步服务器
- **离线可用**：数据缓存在 IndexedDB，断网仍可操作
- **实时协同**：WebSocket 推送增量更新到所有客户端
