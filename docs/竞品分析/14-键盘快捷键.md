## 十四、键盘快捷键完整列表

### 14.1 全局快捷键

| 快捷键 | 功能 |
|--------|------|
| `?` | 打开快捷键帮助 |
| `Cmd/Ctrl + K` | 打开命令面板 |
| `G` then `I` | 跳转到 Inbox |
| `G` then `M` | 跳转到 My Issues |
| `G` then `P` | 跳转到 Projects |
| `G` then `C` | 跳转到 Cycles |
| `G` then `V` | 跳转到 Views |
| `G` then `D` | 跳转到 Documents |
| `G` then `A` | 跳转到 Settings |

### 14.2 Issue 快捷键

| 快捷键 | 功能 |
|--------|------|
| `C` | 创建新 Issue |
| `Cmd/Ctrl + Shift + C` | 从模板创建 Issue |
| `E` | 编辑当前 Issue |
| `D` | 删除当前 Issue |
| `A` | 分配负责人 |
| `L` | 添加标签 |
| `P` | 设置优先级 |
| `S` | 更改状态 |
| `Y` | 复制 Issue 链接 |
| `Enter` | 打开选中的 Issue |
| `Esc` | 关闭 Issue 面板 |

### 14.3 列表视图快捷键

| 快捷键 | 功能 |
|--------|------|
| `J` | 下移一行 |
| `K` | 上移一行 |
| `Shift + J` | 下移并选中 |
| `Shift + K` | 上移并选中 |
| `Cmd/Ctrl + A` | 全选 |
| `Shift + Enter` | 打开选中 Issue |
| `Space` | 切换选中状态 |
| `X` | 选中当前行 |
| `F` | 打开过滤器 |

### 14.4 编辑器快捷键

| 快捷键 | 功能 |
|--------|------|
| `Cmd/Ctrl + B` | 加粗 |
| `Cmd/Ctrl + I` | 斜体 |
| `Cmd/Ctrl + K` | 插入链接 |
| `Cmd/Ctrl + Shift + K` | 插入代码块 |
| `Cmd/Ctrl + L` | 插入列表 |
| `Cmd/Ctrl + Shift + L` | 插入有序列表 |
